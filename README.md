# Sample Login Signup Form

A sample Login/Signup form made with angular 5.



## Features
- Responsive UI.
- JavaScript 
- Form Validation
- No CSS library used
- CI/CD to deploy

## To run locally 

Make sure `node` & `npm` or `yarn` is installed on your system.

```
    > cd sample-login-signup-form/
    > yarn install || npm install
    > yarn run || npm run
```
